Installation

Follow the steps below to setup your site template:

  1. Unzip the downloaded package and open the /portfolio folder to find all the template files. You will need to upload these files to your hosting web server using FTP or localhost in order to use it on your website.
  2. Below is the folder structure and needs to be uploaded to your website or localhost root directory:
    • portfolio/assets - Contains all of the assets referenced
      • portfolio/style - Stylesheet files
      • portfolio/assets/images - Images files
      • portfolio/js - Javacript files
      • portfolio/assets/cv - CV file
    • portfolio/index.html - Homepage
  3. You should upload all or specific HTML files as per your need.
  4. You are good to go for adding your content now!

Helper Classes

Text Size:
  • --p - Changes the Font size to 1rem.
  • --a - Changes the Font size to 1rem.
  • --h1 - Changes the Font size to 2.5rem.
  • --h2 - Changes the Font size to 2rem.
  • --h3 - Changes the Font size to 1.5rem.
Background Color:
  • --primary - Changes the background color to white(#fff).
  • --secondary - Changes the background color to light grey( #a2a2a2).
  • --accent - Changes the background color to light purple(hsl(257, 100%, 60%))
  • --accent2 - Changes the background color to dark purple(hsl(257, 23%, 24%))
Box Shadow
  • --box-shadow -For Regular shadow to elements with box-shadow utilities.( 0 0 12px hsl(0, 0%, 0%))
Text Shadow
  • --text-shadow -For Regular shadow to elements with text-shadow utilities.( 0 3px 0 rgba(255, 255, 255, 0.3))
Borders, Sizing, Spacing (margin & padding) and also much more....
